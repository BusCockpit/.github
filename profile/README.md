# BusCockpit

Open-source hardware cockpit integration for bus simulation games (OMSI 2, Fernbus Simulator).

## Components

| Repo | Language | Purpose |
|------|----------|---------|
| [buscockpit](../buscockpit) | – | Documentation & meta |
| [buslink](../buslink) | Rust | Hardware protocol layer |
| [omsilink](../omsilink) | C# | OMSI 2 plugin |
| [busdriver](../busdriver) | Python | Driver layer |
| [fernbus](../fernbus) | Python | Fernbus Simulator plugin |
| [archive](../archive) | – | Archive |

> Work in progress. Contributions welcome.
